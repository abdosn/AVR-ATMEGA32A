Detailed Description

Interface of GPIO functions for ATMEGA32A microcontroller.

This file contains the interface of GPIO functions, including setting port/pin directions, setting port/pin values, and reading port/pin values.

◆ GPIO_Direction_t

enum GPIO_Direction_t

GPIO direction options.

This enum defines the possible directions for GPIO pins or ports:

  • INPUT_FLOATING: Input without pull-up.
  • INPUT_PULLUP: Input with pull-up enabled.
  • OUTPUT: Output mode.
